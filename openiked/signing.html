OpenIKED Release Signing

OpenIKED portable releases are signed with a signify(1) public key:

RWQpQ9uFY+N0CJU4G4tOjEvD1rj8vJaF1O3BLLrV8ZBc7dScEEYaGq/P

They are co-signed with GPG using this public key. (8E3FC7EC8089D4892BF3DBAAFBC5853596D16718)

Release signatures can be verified using the signify program from within the OpenBSD operating system itself, using the portable version of signify, or by using GPG.
