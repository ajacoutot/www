OpenSMTPD Manual Pages

Web manual pages are available from OpenBSD. These manual pages reflect the latest development release of OpenSMTPD.

  • smtpd(8) — Simple Mail Transfer Protocol daemon
  • smtpd.conf(5) — Simple Mail Transfer Protocol daemon configuration file
  • smtpctl(8) — control the Simple Mail Transfer Protocol daemon
  • newaliases(8) — generate alias mappings for the Simple Mail Transfer Protocol daemon
  • makemap(8) — generate mappings for the Simple Mail Transfer Protocol daemon
  • aliases(5) — aliases files format
  • forward(5) — forwarding files format
  • table(5) — format description for smtpd tables
  • sendmail(8) — a mail enqueuer for smtpd

The SMTP protocol implemented in OpenSMTPD is described in RFC 5321.
